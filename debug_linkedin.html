  1. Custom CloudWatch Metrics protect our customers of our Shopify App 📊 Why? We're running a fraud detection app. That means users can set up rules to block customers. Let's say a customer creates a rule for the name "Sandro Volpicella" from Germany. He can then define an action when this rule matches: ❌ cancel order 🏷️ tag order 🛑 hold fulfillment 🔔 notify team Think about the worst case scenario of our application: Cancelling orders of merchants that shouldn't be cancelled. That means a real lost for the shop. That is why we've created a custom metric for the ActionType Metric: Action Type -> e.g. cancel Dimensions: Shop Based on this metric we have created a CloudWatch Alert. If this alert is in the state ALARM we will be notified. This is mainly to detect bugs very early on.
